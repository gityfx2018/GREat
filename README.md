# GREat@背单词
## this is the Recite helper script that helps to recite words.

Intergrated with GRE-3000 and GRE-Red(6000). Have fun!

## Usage    如何使用:
Use mouse & keyboard [a, s, left, right] to control behavior.
使用鼠标及键盘a, s, 方向左右键控制。当所有单词背完时，此次背结束。

## Notice　注意事项:

当前版本仅供大佬使用。尽情修改吧。
